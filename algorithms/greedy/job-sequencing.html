Greedy Algorithm

Job Sequencing

Intermediate

Schedule jobs with deadlines and profits to maximize total earnings. Sort jobs by profit (descending), then greedily assign each job to the latest available time slot before its deadline.

Time: O(n²)
Optimized: O(n log n)
Space: O(n)
Strategy: Greedy
Optimal: ✓ Yes
💰
Profit First
Sort jobs by profit in descending order so high-value jobs get first pick of available slots.
⏰
Deadline Slots
Each job must complete by its deadline. Time slots t1…tmax represent unit time periods.
🔍
Latest Slot First
For each job, try to assign it to the latest free slot ≤ deadline to leave earlier slots open.
✂️
Skip if Full
If no valid slot is free for a job, skip it. The greedy choice still guarantees maximum profit.
