Greedy Algorithm

Job Sequencing with Deadlines

Intermediate

Given n jobs — each with a deadline and profit — schedule as many as possible (one per time slot) to maximize total profit. Each job takes exactly one unit of time and must finish by its deadline.

Time: O(n²)
Space: O(n)
Type: Greedy
Optimal: ✓ Yes
📊
Sort by Profit
Always consider the highest-profit job first. Greedy choice: maximize profit at each step.
🕐
Latest Slot First
For each job, fill the latest available slot ≤ its deadline. This leaves earlier slots free for other jobs.
🚫
Skip if No Slot
If all slots up to the deadline are already taken, skip this job — no room to schedule it profitably.
⏱️
O(n²) Time
Sorting takes O(n log n). For each of n jobs, we scan up to n slots — giving O(n²) overall.

function jobSequencing(jobs, n): sort(jobs, by profit descending) result = array of size maxDeadline, all -1 for each job in sorted order: for t = job.deadline downto 1: if result[t] == -1: // slot free result[t] = job totalProfit += job.profit break return result // scheduled jobs
