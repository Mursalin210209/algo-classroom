Kruskal's Algorithm

This is a demo implementation page for Kruskal's Algorithm using Greedy strategy.

Demo Pseudocode

Step 1: Sort input based on greedy condition.
Step 2: Pick best available choice.
Step 3: Repeat until solution is complete.

JavaScript Demo (Simple Example)

function greedyDemo(arr) { arr.sort((a,b) => a-b); return arr; } console.log(greedyDemo([5,3,8,1]));
