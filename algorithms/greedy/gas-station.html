Gas Station Queue Simulation

Beginner

A simulation of cars waiting at a gas station using a circular queue for continuous traffic flow. Supports O(1) enqueue (car arrival) and dequeue (serve car) operations. Visualize how vehicles enter, wait, get fueled, and leave with multiple pumps.

Enqueue: O(1)
Dequeue: O(1)
Peek: O(1)
Space: O(n)
🔄
Circular Queue
Fixed-size array used as a queue with wrap-around to avoid shifting elements.
🚗
Enqueue (Arrive)
Add car to rear if not full; update rear = (rear + 1) % size.
⛽
Dequeue (Serve)
Remove car from front if not empty; update front = (front + 1) % size; assign to pump.
🏁
Pump Management
Multiple pumps; cars fuel for a duration then exit, freeing the pump.

💻 Pseudocode
function buildQueue(arr): front = 0; rear = 0; count = 0 for val in arr: enqueue(val)
function enqueue(val): if count == size: return full queue[rear] = val rear = (rear + 1) % size count += 1
function dequeue(): if count == 0: return empty val = queue[front] front = (front + 1) % size count -= 1 return val
