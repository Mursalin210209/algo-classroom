Fractional Knapsack

Detailed step-by-step visualization showing every calculation, comparison, and decision. Each step reveals the mathematical reasoning behind the greedy choice.

Time: O(n log n)
Space: O(n)
Paradigm: Greedy
Optimal: ✓ Yes
⚖️
Value per weight
Ratio = value/weight determines priority. Higher ratio = more valuable per unit weight.
📊
Sorting step
Items are sorted by ratio descending. This is crucial for optimal solution.
🧮
Fraction calculation
When an item doesn't fully fit, fraction = remaining capacity / item weight.
📈
Accumulation
Total value accumulates: whole items add full value, fractions add proportional value.

💻 Pseudocode
function fractionalKnapsack(weights, values, W): // 1. Calculate value/weight ratios items = [] for i in range(n): ratio = values[i] / weights[i] items.append((ratio, weights[i], values[i])) // 2. Sort by ratio descending sort(items, reverse=True) totalValue = 0 remaining = W for (ratio, weight, value) in items: if remaining >= weight: // Take whole item totalValue += value remaining -= weight else: // Take fraction fraction = remaining / weight totalValue += value * fraction remaining = 0 break return totalValue
