Compression Algorithm

Huffman Coding

Intermediate

A prefix-free binary code used for lossless data compression. Builds an optimal variable-length code tree based on symbol frequencies.

Build Time: O(n log n)
Space: O(n)
Optimal: ✓ Yes (greedy)
📊
Frequency Count
Count occurrences of each symbol in the input data.
📉
Priority Queue
Use a min-heap to always select the two least frequent nodes.
🌳
Tree Merging
Combine two trees into a new internal node with summed frequency.
🔑
Code Generation
Traverse the tree to assign binary codes: left=0, right=1.

💻 Pseudocode
function buildHuffmanTree(freqMap): minHeap = createMinHeap from freqMap (nodes with char, freq) while minHeap.size > 1: left = minHeap.popMin() right = minHeap.popMin() newNode = new Node(null, left.freq + right.freq, left, right) minHeap.insert(newNode) return minHeap.popMin() // the root
