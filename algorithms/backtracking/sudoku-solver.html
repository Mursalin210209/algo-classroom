Backtracking Algorithm

Sudoku Solver (Backtracking)

Intermediate

Solve a Sudoku puzzle using backtracking: fill a 9×9 grid with digits 1-9 so each row, column, and 3×3 subgrid contains all digits without repetition.

Worst: O(9!^{9})
Space: O(81)
🔢
Cell Filling
Finds empty cells and tries numbers 1-9.
🔙
Backtracking
If a number leads to conflict, try next number or backtrack.
✅
Safety Check
A number is safe if not in same row, column, or 3x3 box.
🏁
Base Case
If all cells are filled, a solution is found.

💻 Pseudocode
function isSafe(row, col, num, grid): // Check row, col, 3x3 box for num return not in row/col/box function findEmpty(grid): return next empty (row, col) or null function solveSudokuUtil(grid): empty = findEmpty(grid) if not empty: return true row, col = empty for num in 1 to 9: if isSafe(row, col, num, grid): grid[row][col] = num if solveSudokuUtil(grid): return true grid[row][col] = 0 // backtrack return false
