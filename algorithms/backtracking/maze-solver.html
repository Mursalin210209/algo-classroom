Backtracking Algorithm

Maze Solver (Backtracking)

Intermediate

Solve a maze using backtracking: explore paths depth-first, mark visited cells, and backtrack from dead ends until finding the exit or determining no path exists.

Worst: O(4^{m*n})
Space: O(m*n)
🛤️
Path Exploration
Tries directions (down, right, up, left) for each cell, checking validity.
🔙
Backtracking
If a path leads to a dead end, unmark the cell and try another direction.
✅
Safety Check
A move is safe if in bounds, open (0), and not visited.
🏁
Base Case
If the current cell is the exit, a solution is found.

💻 Pseudocode
function isSafe(x, y, maze, visited): if x,y in bounds and maze[x][y] == 0 and not visited[x][y]: return true return false function solveMazeUtil(maze, x, y, visited): if x,y == end: return true visited[x][y] = true for each dir in [down, right, up, left]: nx = x + dx[dir]; ny = y + dy[dir] if isSafe(nx, ny, maze, visited): if solveMazeUtil(maze, nx, ny, visited): return true visited[x][y] = false // backtrack return false
