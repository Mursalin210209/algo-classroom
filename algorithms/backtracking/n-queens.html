Backtracking Algorithm

N-Queens Solver

Intermediate

Place N queens on an N×N chessboard so that no two queens threaten each other. Uses backtracking: try placing a queen, check for conflicts, and backtrack if needed.

Worst: O(N!)
Space: O(N²)
👑
Queen Placement
Place one queen per row, trying each column.
⚔️
Conflict Check
Check if a queen conflicts with previously placed queens (same column or diagonal).
🔙
Backtracking
If no safe column in current row, backtrack to previous row and try next column.
✅
Solution Found
When all N queens are placed successfully.

💻 Pseudocode
function isSafe(board, row, col, n): for i = 0 to row-1: if board[i][col] == 1 return false // same column if col-(row-i) >= 0 and board[i][col-(row-i)] == 1 return false if col+(row-i) < n and board[i][col+(row-i)] == 1 return false return true function solveNQueens(board, row, n): if row == n: return true // all queens placed for col = 0 to n-1: if isSafe(board, row, col, n): board[row][col] = 1 // place queen if solveNQueens(board, row+1, n): return true board[row][col] = 0 // backtrack return false
