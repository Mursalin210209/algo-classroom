Backtracking Algorithm

Permutation Generator (Backtracking)

Intermediate

Generate all permutations of a given array using backtracking by swapping elements and recursing, then backtracking to explore other branches.

Worst: O(n * n!)
Space: O(n)
🔄
Swapping
Swaps elements to fix positions progressively.
🔙
Backtracking
Undoes swaps after exploring a branch.
✅
Base Case
When index reaches end, a permutation is found.
🌳
Recursion Tree
Explores all possible arrangements depth-first.
