Graph Coloring (Backtracking)

Intermediate

Assign colors to graph vertices using backtracking such that no two adjacent vertices share the same color. Tries colors sequentially and backtracks on conflicts.

Worst: O(m^n)
Space: O(n)
🎨
Color Assignment
Tries colors from 1 to m for each vertex, checking safety against neighbors.
🔙
Backtracking
If a color fails for a vertex or subtree, reset and try the next color.
✅
Safety Check
A color is safe if no adjacent vertex has it assigned yet.
🛑
Base Case
If all vertices are colored without conflicts, a solution is found.
