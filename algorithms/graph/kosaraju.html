Kosaraju's Algorithm

Advanced

Find all Strongly Connected Components in a directed graph using two DFS passes. Pass 1 records finish order on the original graph; Pass 2 traverses the transposed graph in reverse finish order.

Time: O(V + E)
Space: O(V + E)
DFS passes: 2
Output: All SCCs
🧲
Strongly Connected
An SCC is a maximal set of vertices where every vertex is reachable from every other vertex — following directed edges.
1️⃣
Pass 1 — Finish Order
Run DFS on the original graph. Push each node onto a stack when it finishes. The last-to-finish node is the best SCC root candidate.
🔄
Transpose the Graph
Reverse all edges. In the transposed graph, SCCs are the same sets — but cross-SCC edges point in the opposite direction.
2️⃣
Pass 2 — Pop & DFS
Pop nodes from the finish stack. Run DFS on the transposed graph from each unvisited popped node — each DFS tree is one SCC.

💻 Pseudocode
function Kosaraju(G): # Pass 1: DFS on original graph visited = {}; finStack = [] for v in V: if v not in visited: dfs1(G, v, visited, finStack) # Transpose: reverse all edges GT = transpose(G) # Pass 2: DFS on transposed graph visited2 = {}; sccs = [] while finStack not empty: v = finStack.pop() if v not in visited2: scc = [] dfs2(GT, v, visited2, scc) sccs.append(scc) # one SCC! return sccs
