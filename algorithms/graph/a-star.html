A* Search Algorithm

Intermediate

An informed search algorithm that finds the shortest path using a heuristic to guide the search towards the goal.

Time: O((V + E) log V)
Space: O(V)
Shortest path: ✓ Guaranteed (with admissible heuristic)
Structure: Priority Queue
⭐
Heuristic Guidance
Uses f = g + h, where g is path cost, h is estimated cost to goal.
📊
Priority Queue
Expands the most promising node based on f-score.
📍
Optimal Path
Guarantees shortest path if heuristic is admissible and consistent.
⏱️
O((V + E) log V)
Efficient with binary heap; often faster than Dijkstra in practice.

💻 Pseudocode
function AStar(graph, start, target, h): gScore = {} # All ∞ except gScore[start] = 0 fScore = {} # All ∞ except fScore[start] = h(start) prev = {} pq = new PriorityQueue() # Min-heap by fScore pq.insert(start, fScore[start]) while pq not empty: current = pq.extractMin() if current == target: path = reconstructPath(prev, target) return path, gScore[target] for neighbor, weight in graph.adj[current]: tent_g = gScore[current] + weight if tent_g < gScore[neighbor]: gScore[neighbor] = tent_g fScore[neighbor] = tent_g + h(neighbor) prev[neighbor] = current pq.decreaseKey(neighbor, fScore[neighbor]) return null # Not found
