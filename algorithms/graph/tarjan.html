Tarjan's Algorithm

Advanced

Find all Strongly Connected Components in a single DFS pass. Each node tracks its discovery time and a low-link value — the smallest discovery time reachable from its subtree. When a node's low-link equals its discovery time, it's the root of an SCC.

Time: O(V + E)
Space: O(V)
DFS passes: 1
vs Kosaraju: 1 pass, 1 graph
Output: All SCCs
🔢
Discovery Time
Each node gets a unique timestamp when first visited. disc[u] = the DFS visit counter at the moment u is entered.
🔗
Low-Link Value
low[u] = the minimum discovery time reachable from the subtree rooted at u (via back-edges or tree-edges). Updated on DFS return.
📚
Tarjan Stack
Nodes are pushed onto a stack when visited and only popped when their SCC is identified. onStack[] tracks membership.
🌱
SCC Root Condition
When DFS returns to node u and low[u] == disc[u], u is the root of an SCC. Pop the stack down to u — those nodes form the SCC.

💻 Pseudocode
var timer=0, disc[], low[], onStack[], stack=[] function dfs(u): disc[u] = low[u] = timer++ stack.push(u); onStack[u] = true for v in adj[u]: if disc[v] == -1: # unvisited dfs(v) low[u] = min(low[u], low[v]) elif onStack[v]: # back-edge low[u] = min(low[u], disc[v]) # SCC root check if low[u] == disc[u]: scc = [] while stack.top != u: w = stack.pop(); onStack[w]=false scc.append(w) stack.pop(u); scc.append(u) sccs.append(scc)
