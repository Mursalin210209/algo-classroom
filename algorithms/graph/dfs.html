Graph Algorithm

Depth-First Search

Intermediate

Dive deep into graph branches before backtracking. Uses a stack to explore paths, but no shortest path guarantee in unweighted graphs.

Time: O(V + E)
Space: O(V)
Shortest path: ✗ Not Guaranteed
Structure: Stack (LIFO)
🕳️
Depth Dive
DFS goes as deep as possible along a branch before backtracking to explore others.
📚
Stack — LIFO
A stack (Last-In, First-Out) ensures we process the most recently discovered nodes first.
📍
Path Finding
DFS finds a path to the target, but it may not be the shortest one due to deep exploration.
⏱️
O(V + E) Time
Each vertex and edge is visited once. Space complexity is also O(V) for the stack and visited set.
