Graph Algorithm

Breadth-First Search

Intermediate

Explore a graph level by level — all nodes at distance 1, then distance 2, and so on. Uses a queue to guarantee shortest paths in unweighted graphs.

Time: O(V + E)
Space: O(V)
Shortest path: ✓ Guaranteed
Structure: Queue (FIFO)
🌊
Wave Expansion
BFS spreads out like ripples — visits all neighbors at distance 1 before distance 2, and so on.
📬
Queue — FIFO
A queue (First-In, First-Out) ensures we process nodes in the order they were discovered.
📍
Shortest Path
BFS always finds the shortest path in an unweighted graph — guaranteed by level-by-level exploration.
⏱️
O(V + E) Time
Each vertex and edge is visited once. Space complexity is also O(V) for the queue and visited set.

💻 Pseudocode
function BFS(graph, start, target): queue = new Queue() visited = new Set() parent = {} queue.enqueue(start) visited.add(start) parent[start] = null while queue not empty: current = queue.dequeue() if current == target: path = reconstructPath(parent, target) return path for neighbor in graph.adj[current]: if neighbor not in visited: visited.add(neighbor) parent[neighbor] = current queue.enqueue(neighbor) return null # Not found
