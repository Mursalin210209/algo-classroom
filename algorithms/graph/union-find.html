Graph / Set Algorithm

Union-Find

Intermediate

Maintain a partition of elements into disjoint sets. Supports two operations: Find (which set contains x?) and Union (merge two sets). With path compression and union by rank, each operation is nearly O(1).

Time (amortized): O(α(n))
Space: O(n)
Path Compression: ✓
Union by Rank: ✓
🌲
Forest of Trees
Each disjoint set is represented as a tree. The root is the canonical representative of the set.
🔍
Find with Compression
Traverses up to the root, then flattens the path — making future finds nearly O(1).
🔗
Union by Rank
Always attach the shorter tree under the taller tree's root to keep trees shallow.
⚡
α(n) Complexity
The inverse Ackermann function — effectively constant for any realistic input size. Nearly O(1) per operation.

function find(x): if parent[x] != x: parent[x] = find(parent[x]) // path compress return parent[x] function union(a, b): ra = find(a); rb = find(b) if ra == rb: return // same set if rank[ra] < rank[rb]: parent[ra] = rb else if rank[ra] > rank[rb]: parent[rb] = ra else: parent[rb] = ra rank[ra] += 1
