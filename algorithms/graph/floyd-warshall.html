Graph Algorithm · Dynamic Programming

Floyd-Warshall

Advanced

Find shortest paths between all pairs of vertices simultaneously. Uses dynamic programming — progressively improving estimates by considering each vertex as an intermediate.

Time: O(V³)
Space: O(V²)
Negative edges: ✓ Supported
All-pairs: ✓ Yes
Neg. cycles: ✓ Detected
🔮
All-Pairs Shortest Path
Unlike BFS or Dijkstra, Floyd-Warshall finds shortest paths between every pair of vertices at once.
🔗
Intermediate Vertex
For each k, we ask: "Is going through k a shorter route?" The key DP recurrence: dist[i][j] = min(dist[i][j], dist[i][k] + dist[k][j]).
➖
Negative Edges OK
Floyd-Warshall handles negative-weight edges correctly — unlike Dijkstra. Negative cycles are detected if any dist[i][i] < 0.
📊
O(V³) Time
Three nested loops over all vertices. The V×V distance matrix is updated in-place across V intermediate vertices.
