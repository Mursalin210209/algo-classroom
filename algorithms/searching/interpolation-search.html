Searching Algorithm

Interpolation Search

Beginner

First sorts the array if necessary, then finds the position of a target value within the sorted array by estimating the position based on the values at the bounds.

Best: O(1)
Average: O(log log n)
Worst: O(n)
Space: O(1)
📏
Sorted Array
Requires the input array to be sorted in ascending order.
📐
Position Estimation
Estimates the likely position of the target using linear interpolation.
🔄
Adaptive Probing
Adjusts the probe position based on the target value and bounds.
⚡
Efficient for Uniform
Performs best when values are uniformly distributed.

💻 Pseudocode
function interpolationSearch(arr, target): low = 0 high = arr.length - 1 while low <= high and target >= arr[low] and target <= arr[high]: if low == high: if arr[low] == target: return low return -1 pos = low + floor(((high - low) / (arr[high] - arr[low])) * (target - arr[low])) if arr[pos] == target: return pos if arr[pos] < target: low = pos + 1 else: high = pos - 1 return -1
