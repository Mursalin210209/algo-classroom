Searching Algorithm

Exponential Search

Intermediate

Finds a range where the element may exist by doubling the index exponentially, then performs a Binary Search within that specific range. Very efficient for unbounded or large arrays.

Best: O(1)
Average: O(log i)
Worst: O(log i)
Space: O(1)
📈
Range Doubling
Jumps by powers of 2 (1, 2, 4, 8...) to find a boundary.
🎯
Binary Phase
Once a range is found, switches to Binary Search.
📏
Sorted Only
Requires the input array to be sorted for the logic to hold.
♾️
Large Lists
Excellent for datasets where the target is near the start.

💻 Pseudocode
function exponentialSearch(arr, target):   if arr[0] == target: return 0   i = 1   while i < n and arr[i] <= target:     i = i * 2   return binarySearch(arr, target, i/2, min(i, n-1))
