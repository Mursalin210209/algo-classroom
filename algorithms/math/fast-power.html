Fast Power (Binary Exponentiation)

Intermediate

Compute aⁿ efficiently by using the binary representation of the exponent. Instead of multiplying n times (O(n)), we process bits from right to left, squaring the base at each step and multiplying when we see a 1 bit.

Time: O(log n)
Space: O(1)
Method: Exponentiation by Squaring
🔢
Binary Representation
The exponent n is written in binary. Each 1 bit tells us to multiply the current power into our result.
🔄
Repeated Squaring
At each step, we square the base: x → x² → x⁴ → x⁸ ... building powers of 2.
⚡
Logarithmic Time
Instead of n multiplications, we only need O(log n) multiplications — much faster for large exponents.
🧮
Modular Exponentiation
Often used in cryptography with modulo arithmetic to keep numbers manageable.
