Fast Power

Intermediate

Compute baseexp (optionally mod m) in O(log exp) steps by repeatedly squaring. Also called Binary Exponentiation or Exponentiation by Squaring.

Time: O(log exp)
Space: O(1)
Naive: O(exp) multiplications
Key idea: exp → binary bits
🔢
Core Idea
Write exp in binary. Each bit tells us whether to multiply the current base power into the result.
✂️
Halving Trick
b^n = (b²)^(n/2). Each iteration squares the base and halves the exponent — logarithmic iterations.
🔵
Odd Exponent
When exp is odd, multiply result by the current base before halving. Handles every set bit in binary.
🔐
Modular Variant
Apply mod m at each step to keep numbers small — critical for cryptography and competitive programming.

function fastPow(base, exp, mod=0): result = 1 if mod: base = base % mod while exp > 0: if exp % 2 == 1: // odd bit result *= base if mod: result %= mod base *= base // square if mod: base %= mod exp >>= 1 // halve return result
