Number Theory

Modular Inverse

Intermediate

Find x such that a · x ≡ 1 (mod m). The modular inverse exists if and only if gcd(a, m) = 1. Visualize both the Extended Euclidean method and Fermat's Little Theorem.

Extended GCD: O(log min(a,m))
Fermat: O(log m)
Exists when: gcd(a,m) = 1
Fermat requires: m is prime
🔄
Definition
a⁻¹ mod m is the number x where a·x ≡ 1 (mod m). Like division in modular arithmetic.
🔗
Extended GCD
Run Euclidean algorithm, then back-substitute to express 1 = a·x + m·y. The x is your inverse.
⚡
Fermat's Little Theorem
If m is prime: a⁻¹ ≡ a^(m−2) mod m. Uses fast modular exponentiation.
🚫
No Inverse Case
If gcd(a, m) > 1, the inverse doesn't exist — a and m share a common factor, so division is undefined.

💻 Pseudocode EXT GCD
function extGCD(a, b): if b == 0: return (a, 1, 0) (g, x1, y1) = extGCD(b, a % b) x = y1 y = x1 - ⌊a/b⌋ · y1 return (g, x, y) function modInverse(a, m): (g, x, _) = extGCD(a, m) if g != 1: return None // no inverse return (x % m + m) % m
