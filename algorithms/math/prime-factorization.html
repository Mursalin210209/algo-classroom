Number Theory

Prime Factorisation

Beginner

Decompose any integer n into a product of prime numbers using Trial Division. Try each divisor from 2 up to √n, dividing out every occurrence.

Time: O(√n)
Space: O(log n)
Unique by: Fundamental Theorem
Works for: n ≥ 2
🏗️
Trial Division
Try each integer d from 2 upward. While d divides n, record d as a factor and divide n by d.
√
√n Bound
We only need to check divisors up to √n. If nothing divides by then, the remaining n is itself prime.
🔑
Fundamental Theorem
Every integer > 1 has a unique prime factorisation — order doesn't matter, the set of primes is fixed.
⚡
Optimisation
Check 2 separately, then only odd divisors. Reduces work by ~half vs checking every integer.
