Sorting Algorithm

Insertion Sort

Beginner

Builds a sorted subarray on the left by taking each new element and inserting it into the correct position by swapping it leftward until it's in place.

Best: O(n)
Average: O(n²)
Worst: O(n²)
Space: O(1)
Stable: ✓ Yes
🔄
Adjacent Swaps
Swaps the current element leftward with adjacent predecessors until sorted.
🫧
Bubbling Down
The current element "bubbles" left to its correct position in the sorted prefix.
⚡
Early Stop
For each insertion, stops comparisons early if the element is already in place — O(n) for sorted arrays.
📚
Stable Sort
Equal elements maintain their original relative order — a useful property for multi-key sorting.

💻 Pseudocode
function insertionSort(arr): n = arr.length for i = 1 to n-1: j = i while j > 0 and arr[j-1] > arr[j]: swap(arr[j-1], arr[j]) j = j - 1 return arr
