Sorting Algorithm

Insertion Sort

Beginner

Builds a sorted subarray on the left by taking each new element and inserting it into the correct position by swapping it leftward until it's in place.

Best: O(n)
Average: O(n²)
Worst: O(n²)
Space: O(1)
Stable: ✓ Yes
🃏
Like Playing Cards
Think of sorting a hand of cards - you take one card and insert it into the correct position among the already-sorted cards.
⬅️
Shifting Left
Each new element is shifted left, swapping with larger neighbors until it finds its proper place.
⚡
Adaptive
For nearly-sorted arrays, insertion sort runs in nearly O(n) time - great for partially sorted data!
📚
Online Algorithm
Can sort elements as they arrive - you don't need the entire array upfront.

💻 Pseudocode
function insertionSort(arr): n = arr.length for i = 1 to n-1: j = i while j > 0 and arr[j-1] > arr[j]: swap(arr[j-1], arr[j]) j = j - 1 return arr
