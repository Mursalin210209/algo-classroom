Sorting Algorithm

Bubble Sort

Beginner

Repeatedly compare adjacent elements and swap them if they're in the wrong order. After each full pass, the largest unsorted element "bubbles up" to its correct position.

Best: O(n)
Average: O(n²)
Worst: O(n²)
Space: O(1)
Stable: ✓ Yes
🔄
Adjacent Swaps
Each pass compares neighbors and swaps out-of-order pairs. Only adjacent elements ever swap.
🫧
Bubbling Up
The largest unsorted element moves to its correct position at the end after each full pass.
⚡
Early Exit
If no swaps happen during a pass, the array is sorted — O(n) best case for already-sorted arrays.
📚
Stable Sort
Equal elements maintain their original relative order — a useful property for multi-key sorting.

💻 Pseudocode
function bubbleSort(arr): n = arr.length for i = 0 to n-2: swapped = false for j = 0 to n-i-2: if arr[j] > arr[j+1]: swap(arr[j], arr[j+1]) swapped = true if not swapped: break return arr
