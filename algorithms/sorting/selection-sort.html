Sorting Algorithm

Selection Sort

Beginner

Repeatedly select the smallest unsorted element and swap it to the front of the unsorted portion. Builds the sorted list from the beginning.

Best: O(n²)
Average: O(n²)
Worst: O(n²)
Space: O(1)
Stable: ✗ No
🔍
Min Selection
In each pass, scan the unsorted portion to find the smallest element.
🔄
Swap to Front
Swap the found minimum with the first unsorted position.
📈
Fixed Passes
Always performs exactly n-1 passes with O(n²) comparisons — no early termination.
⚠️
Unstable Sort
Equal elements may not maintain their relative order after sorting.

💻 Pseudocode
function selectionSort(arr): n = arr.length for i = 0 to n-2: minIdx = i for j = i+1 to n-1: if arr[j] < arr[minIdx]: minIdx = j if minIdx != i: swap(arr[i], arr[minIdx]) return arr
