Sorting Algorithm

Selection Sort

Beginner

Repeatedly find the smallest element in the unsorted portion and swap it to the front. After each pass, one more element sits in its final sorted position.

Best: O(n²)
Average: O(n²)
Worst: O(n²)
Space: O(1)
Stable: ✗ No (typical)
🎯
Find Minimum
Scan the unsorted section to locate the smallest element.
🔀
Swap to Front
Swap the minimum element with the first unsorted position, extending the sorted region.
📈
Quadratic time
Even if array is sorted, it still does O(n²) comparisons (but minimal swaps).
⚡
In‑place
Only uses a constant amount of extra memory (O(1)).

💻 Pseudocode
function selectionSort(arr): n = arr.length for i = 0 to n-2: minIdx = i for j = i+1 to n-1: if arr[j] < arr[minIdx]: minIdx = j if minIdx != i: swap(arr[i], arr[minIdx]) return arr
