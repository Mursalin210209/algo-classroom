Sorting Algorithm

Counting Sort

Beginner

Sorts elements by counting the number of occurrences of each unique key value. Efficient when the range of values is not significantly larger than the number of elements.

Best: O(n+k)
Average: O(n+k)
Worst: O(n+k)
Space: O(n+k)
Stable: ✓ Yes
🔢
Counting Phase
Count the occurrences of each value in the input array.
📈
Accumulation
Convert counts to cumulative sums to determine ending positions.
📤
Placement Phase
Place elements into the output array in reverse order to maintain stability.
📚
Stable Sort
Equal elements maintain their original relative order — a useful property for multi-key sorting.

💻 Pseudocode
function countingSort(arr): n = arr.length min = min(arr) max = max(arr) range = max - min + 1 count = new Array(range).fill(0) for i = 0 to n-1: count[arr[i] - min] += 1 for i = 1 to range-1: count[i] += count[i-1] output = new Array(n) for i = n-1 downto 0: key = arr[i] - min pos = count[key] - 1 output[pos] = arr[i] count[key] -= 1 return output
