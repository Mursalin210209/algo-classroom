Sorting Algorithm

Counting Sort

Beginner

Sort integers in O(n+k) time by counting occurrences of each value, computing prefix sums, then placing each element into its correct output position — no comparisons needed.

Best: O(n+k)
Average: O(n+k)
Worst: O(n+k)
Space: O(n+k)
Stable: ✓ Yes
🔢
Count Occurrences
First pass tallies how many times each value appears, storing counts in an auxiliary count array.
➕
Prefix Sums
Transform counts into prefix sums so each entry tells us the last position that value should occupy.
🎯
Direct Placement
Traverse input right-to-left, placing each element at the exact index given by the prefix sum array.
⚡
No Comparisons
Counting Sort never compares two elements — it beats O(n log n) when k (value range) is small.

💻 Pseudocode
function countingSort(arr): k = max(arr) + 1 count = new Array(k).fill(0) for each val in arr: count[val]++ for i = 1 to k-1: count[i] += count[i-1] output = new Array(n) for i = n-1 downto 0: output[--count[arr[i]]] = arr[i] return output
