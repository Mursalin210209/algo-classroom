Merge Sort

Intermediate

A divide-and-conquer algorithm that recursively splits the array in half, sorts each half, then merges the sorted halves back together. Guaranteed O(n log n) in all cases.

Best: O(n log n)
Average: O(n log n)
Worst: O(n log n)
Space: O(n)
Stable: ✓ Yes
✂️
Divide
Recursively split the array at the midpoint until every sub-array has only one element (already sorted).
🔀
Conquer
Merge pairs of sorted sub-arrays, always picking the smaller of the two front elements each time.
📐
Log Depth
Splitting halves creates log₂(n) levels. Each level does O(n) work, giving guaranteed O(n log n) total.
🏗️
Stable & External
Equal elements keep their original order. Works great for linked lists and external (disk-based) sorting.

💻 Pseudocode
function mergeSort(arr): if arr.length <= 1: return arr mid = arr.length / 2 left = mergeSort(arr[0..mid]) right = mergeSort(arr[mid..]) return merge(left, right) function merge(left, right): result = [] while left and right not empty: if left[0] <= right[0]: result.push(left.shift()) else: result.push(right.shift()) return result + left + right
