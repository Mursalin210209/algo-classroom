Sorting Algorithm

Merge Sort

Intermediate

Recursively divide the array into halves, sort each half, and merge the sorted halves back together while maintaining order.

Best: O(n log n)
Average: O(n log n)
Worst: O(n log n)
Space: O(n)
Stable: ✓ Yes
🔪
Divide & Conquer
Recursively split the array into smaller subarrays until base case.
🔀
Merge Step
Combine two sorted subarrays by comparing and picking smaller elements.
📚
Stable Sort
Preserves relative order of equal elements, useful for multi-key sorts.
💾
Auxiliary Space
Requires O(n) extra space for temporary storage during merging.

💻 Pseudocode
function mergeSort(arr, left, right): if left >= right: return mid = floor((left + right) / 2) mergeSort(arr, left, mid) mergeSort(arr, mid + 1, right) merge(arr, left, mid, right) function merge(arr, left, mid, right): aux = new Array(right - left + 1) for k = left to right: aux[k] = arr[k] i = left, j = mid + 1, k = left while i <= mid and j <= right: if aux[i] <= aux[j]: arr[k] = aux[i] i++, k++ else: arr[k] = aux[j] j++, k++ while i <= mid: arr[k] = aux[i]; i++, k++ while j <= right: arr[k] = aux[j]; j++, k++
