Sorting Algorithm

Merge Sort

Intermediate

A divide-and-conquer algorithm that recursively divides the array into halves, sorts the halves, and merges them back into a sorted array.

Best: O(n log n)
Average: O(n log n)
Worst: O(n log n)
Space: O(n)
Stable: ✓ Yes
🔀
Divide & Conquer
Breaks the array into smaller subproblems, solves them recursively, and combines the solutions.
🌳
Recursion Tree
The process forms a binary tree of divisions down to base cases, then merges up.
⚡
Efficient Merging
Combines two sorted halves in linear time using two pointers.
📚
Stable Sort
Preserves the relative order of equal elements, useful for sorting by multiple keys.

function mergeSort(arr): if arr.length <= 1: return arr mid = floor(arr.length / 2) left = mergeSort(arr.slice(0, mid)) right = mergeSort(arr.slice(mid)) return merge(left, right)
