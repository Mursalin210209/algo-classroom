Sorting Algorithm

Shell Sort

Intermediate

A generalization of Insertion Sort that sorts elements far apart first using a gap sequence, then progressively reduces the gap — making distant swaps cheap and final passes nearly sorted.

Best: O(n log n)
Average: O(n log² n)
Worst: O(n²)
Space: O(1)
Stable: ✗ No
📐
Gap Sequence
We use a shrinking gap (n/2, n/4, … 1). The gap controls how far apart elements are when compared and swapped.
🚀
Long-Distance Swaps
Large gaps let small elements jump far left in one step — avoiding the slow one-at-a-time crawl of plain insertion sort.
🔄
Diminishing Passes
Each pass (with a smaller gap) refines the order. By the time gap=1, the array is nearly sorted, so that pass is very fast.
📦
In-Place
Like insertion sort, no extra memory is needed. All sorting happens within the original array — O(1) space.

💻 Pseudocode
function shellSort(arr): n = arr.length gap = floor(n / 2) while gap > 0: for i = gap to n-1: j = i while j >= gap and arr[j-gap] > arr[j]: swap(arr[j-gap], arr[j]) j -= gap gap = floor(gap / 2) return arr
