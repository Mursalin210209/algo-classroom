💻 Pseudocode
function radixSort(arr): maxVal = max(arr) digits = floor(log10(maxVal)) + 1 place = 1 for i = 0 to digits-1: containers = [[], [], [], [], [], [], [], [], [], []] for num in arr: digit = floor(num / place) % 10 containers[digit].push(num) arr = [].concat(...containers) place *= 10 return arr
