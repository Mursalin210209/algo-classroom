String Algorithm

Rabin-Karp

Intermediate

String matching algorithm that uses hashing to efficiently search for a pattern in text by computing rolling hashes.

Time: Avg O(n+m), Worst O(nm)
Space: O(1)
🔄
Rolling Hash
Computes hash for sliding windows efficiently without recomputing from scratch.
📊
Pattern Hash
Precomputed hash value for the entire pattern.
⚠️
Collision Check
Verifies exact match when hashes collide to avoid false positives.
🚀
Efficient Search
Average linear time for searching, suitable for multiple patterns.

💻 Pseudocode
function rabinKarp(text, pattern, q): d = 256 // base n = text.length m = pattern.length p = 0 // pattern hash t = 0 // text hash h = 1 for i = 1 to m-1: h = (h * d) % q for i = 0 to m-1: p = (d * p + ord(pattern[i])) % q t = (d * t + ord(text[i])) % q for s = 0 to n-m: if p == t: match = true for j = 0 to m-1: if text[s+j] != pattern[j]: match = false; break if match: found at s if s < n-m: t = (d * (t - ord(text[s]) * h) + ord(text[s+m])) % q if t < 0: t += q
