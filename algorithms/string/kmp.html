String Algorithm

Knuth-Morris-Pratt

Intermediate

Efficient string matching algorithm that preprocesses the pattern to create a partial match table (pi), allowing it to skip unnecessary comparisons in the text.

Time: O(n + m)
Space: O(m)
📊
Prefix Table
Computes the longest proper prefix that is also a suffix for each substring of the pattern.
🔄
Shift on Mismatch
Uses the prefix table to determine how far to shift the pattern when a mismatch occurs.
🚫
No Backtracking
Never re-compares characters in the text, making it efficient for large texts.
⚡
Linear Time
Total time is linear in the lengths of the text and pattern.

💻 Pseudocode
function computePi(pattern): m = pattern.length pi = new Array(m).fill(0) k = 0 for q = 1 to m-1: while k > 0 and pattern[k] != pattern[q]: k = pi[k-1] if pattern[k] == pattern[q]: k += 1 pi[q] = k function kmpSearch(text, pattern): pi = computePi(pattern) q = 0 for i = 0 to text.length-1: while q > 0 and pattern[q] != text[i]: q = pi[q-1] if pattern[q] == text[i]: q += 1 if q == pattern.length: match at i - q + 1 q = pi[q-1]
