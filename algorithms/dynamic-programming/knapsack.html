Dynamic Programming

0/1 Knapsack Problem

Intermediate

Given a set of items — each with a weight and value — fill a knapsack of fixed capacity to maximize total value. Each item can be taken (1) or left (0). Solved optimally with a 2D DP table.

Time: O(n·W)
Space: O(n·W)
Type: Bottom-up DP
Optimal: ✓ Yes
📦
Subproblems
dp[i][w] = max value using first i items with capacity w. Build answer from smaller subproblems.
⚖️
Include or Skip
For each item, decide: skip it (keep previous value) or include it (if it fits) and take the better choice.
🔄
Recurrence
dp[i][w] = max(dp[i-1][w], val[i] + dp[i-1][w-wt[i]]) if wt[i] ≤ w, else dp[i-1][w].
🔍
Traceback
After filling the table, trace back from dp[n][W] to find exactly which items were selected.

💻 Pseudocode
function knapsack(wt, val, W, n): dp = (n+1) × (W+1) table of 0s for i = 1 to n: for w = 0 to W: if wt[i] > w: dp[i][w] = dp[i-1][w] else: dp[i][w] = max(dp[i-1][w], val[i] + dp[i-1][w-wt[i]]) return dp[n][W] // traceback for items
