Dynamic Programming

Subset Sum Problem

Intermediate

Given a set of positive integers and a target sum, determine if there is a subset that sums exactly to the target. Solved with a 2D DP table.

Time: O(m·target)
Space: O(m·target)
Type: Bottom-up DP
Optimal: ✓ Yes (for small target)
📦
Subproblems
dp[i][s] = true if subset of first i numbers sums to s.
⚖️
Include or Skip
dp[i][s] = dp[i-1][s] (skip) or dp[i-1][s - num[i]] (include, if possible).
🔄
Recurrence
dp[i][s] = dp[i-1][s] || (s >= num[i] ? dp[i-1][s - num[i]] : false).
🔍
Traceback
Trace back from dp[m][target] to find the exact subset if possible.

💻 Pseudocode
function subsetSum(arr, target, m): dp = (m+1) × (target+1) false dp[0][0] = true for i = 1 to m: for s = 0 to target: dp[i][s] = dp[i-1][s] if s >= arr[i-1]: dp[i][s] = dp[i][s] || dp[i-1][s - arr[i-1]] return dp[m][target] // traceback for subset
