Dynamic Programming

Longest Palindromic Subsequence

Intermediate

Given a string, find the length of its longest subsequence that reads the same forwards and backwards. Unlike substrings, subsequences don't need to be contiguous — characters can be skipped.

Time: O(n²)
Space: O(n²)
Type: Interval DP
Fill: Diagonal by diagonal
🔤
Subsequence
Characters don't need to be adjacent. "bab" is a subsequence of "bbab" by picking characters at indices 0, 1, 3.
🪞
Palindrome Check
If s[i] == s[j], then dp[i][j] = dp[i+1][j-1] + 2. The outer characters extend any inner palindrome.
↔️
Mismatch Case
If s[i] != s[j], then dp[i][j] = max(dp[i+1][j], dp[i][j-1]). Skip one end and take the best.
🔍
Traceback
Walk the table from dp[0][n-1] back to reconstruct which characters form the palindrome.

function lps(s): n = s.length dp[i][i] = 1 // single char for L = 2 to n: for i = 0 to n-L: j = i + L - 1 if s[i] == s[j]: dp[i][j] = dp[i+1][j-1] + 2 else: dp[i][j] = max(dp[i+1][j], dp[i][j-1]) return dp[0][n-1]
