Dynamic Programming

Rod Cutting Problem

Intermediate

Given a rod of length n and prices for pieces of lengths 1 to n, find the maximum revenue obtainable by cutting up the rod and selling the pieces. Solved with 1D DP.

Time: O(n²)
Space: O(n)
Type: Bottom-up DP
Optimal: ✓ Yes
📦
Subproblems
dp[i] = max revenue for rod of length i. Build from smaller lengths.
⚖️
Cut or Not
For each possible first cut j, revenue = price[j] + dp[i-j]. Take max over j=1 to i.
🔄
Recurrence
dp[i] = max(price[j] + dp[i-j]) for j=1 to i. Base: dp[0]=0.
🔍
Traceback
Trace back to find the optimal cuts that achieve max revenue.

💻 Pseudocode
function rodCut(price, n): dp = array of 0s size n+1 for i = 1 to n: for j = 1 to i: dp[i] = max(dp[i], price[j] + dp[i-j]) return dp[n] // traceback for cuts
