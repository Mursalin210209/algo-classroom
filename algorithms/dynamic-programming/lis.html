Dynamic Programming

Longest Increasing Subsequence

Intermediate

Given an array of integers, find the length of the longest strictly increasing subsequence. Solved with O(n²) dynamic programming (note: faster O(n log n) methods exist).

Time: O(n²)
Space: O(n)
Type: Bottom-up DP
Optimal: No (O(n log n) possible)
📏
Subproblems
dp[i] = length of LIS ending at index i. Build from previous elements.
🔄
Update or Skip
For each previous j, if arr[j] < arr[i], update dp[i] = max(dp[i], dp[j] + 1).
📜
Recurrence
dp[i] = 1 + max(dp[j] for j < i where arr[j] < arr[i]), else 1.
🔙
Traceback
From max dp value, backtrack to reconstruct one possible LIS.

💻 Pseudocode
function lis(arr, n): dp = array of 1s size n for i = 1 to n-1: for j = 0 to i-1: if arr[j] < arr[i] and dp[j]+1 > dp[i]: dp[i] = dp[j] + 1 return max(dp) // traceback for sequence
