Data Structure

Linked List

Beginner

A linear data structure where each node holds a value and a pointer to the next node. In a doubly linked list, each node also holds a pointer back to the previous node. Watch exactly how pointers are rewired during insert and delete.

Insert Head: O(1)
Insert Tail: O(n)
Insert After: O(n)
Delete: O(n)
Search: O(n)
Space: O(n)
📦
Node Structure
Each node stores a value and one (singly) or two (doubly) pointers to neighbouring nodes.
➡️
Next Pointer
Points to the next node in the chain. The last node's next is null (∅).
⬅️
Prev Pointer (DLL)
Doubly linked lists also track the previous node, enabling O(1) backward traversal.
🔄
Pointer Rewiring
Insert/delete only changes a constant number of pointers — no shifting like arrays.
