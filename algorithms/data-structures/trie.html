Data Structure

Trie (Prefix Tree)

Intermediate

A tree-like data structure for storing strings where each node represents a single character. Nodes share common prefixes, making prefix search and autocomplete extremely efficient.

Search: O(m)
Insert: O(m)
Prefix: O(m)
Space: O(n·m)
m = word length
📖
Prefix Sharing
Words with common prefixes share the same path from root, saving space and enabling fast lookups.
➕
Insertion
Walk character by character; create nodes for missing characters; mark end-of-word at the last node.
🔍
Search
Follow the path character by character; return true only if the final node is marked as end-of-word.
💡
Autocomplete
Find the prefix node, then DFS all descendants — every end-of-word path is a valid completion.

💻 Pseudocode
function insert(root, word): node = root for char in word: if char not in node.children: node.children[char] = new Node() node = node.children[char] node.isEnd = true
function search(root, word): node = root for char in word: if char not in node.children: return false node = node.children[char] return node.isEnd
function startsWith(root, prefix): node = root for char in prefix: if char not in node.children: return false node = node.children[char] return true
