Queue Operations

Beginner

A Queue is a First-In, First-Out (FIFO) data structure. Elements are added to the rear and removed from the front. Visualize Enqueue, Dequeue, Peek, and isEmpty operations step by step.

Enqueue: O(1)
Dequeue: O(1)
Peek: O(1)
Search: O(n)
Space: O(n)
➕
Enqueue
Add an element to the rear of the queue. The new element becomes the new rear.
➖
Dequeue
Remove and return the front element. The element after it becomes the new front.
👁️
Peek / Front
Look at the front element without removing it. Useful for checking what comes next.
📋
FIFO Principle
First In, First Out — like a queue of people. The first one added is the first one removed.
🧑‍🏫
Welcome to Queue Operations! A queue works like a line of people — you can only add to the rear or remove from the front. Let's explore it step by step.

💻 Pseudocode
class Queue: queue = [] def enqueue(val): queue.append(val) def dequeue(): if isEmpty(): return null return queue.pop(0) def peek(): if isEmpty(): return null return queue[0] def isEmpty(): return queue.length == 0
