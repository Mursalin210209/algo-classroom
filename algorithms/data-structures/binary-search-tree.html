Data Structure

Binary Search Tree

Intermediate

A binary tree where for each node, all elements in the left subtree are less than the node, and all in the right subtree are greater. Supports efficient search, insertion, and deletion.

Search: O(log n)
Insert: O(log n)
Delete: O(log n)
Space: O(n)
Balanced: ✗ No (can skew)
🔍
Efficient Search
Halve the search space at each step by going left or right based on comparisons.
➕
Insertion
Traverse like search; insert as leaf where value fits the BST property.
🗑️
Deletion
Handle cases: no child, one child, or two children (replace with successor).
⚖️
Balance Issues
Worst-case O(n) if insertions are sorted; self-balancing variants like AVL fix this.

💻 Pseudocode
function insert(root, val): if root is null: return new Node(val) if val < root.val: root.left = insert(root.left, val) else if val > root.val: root.right = insert(root.right, val) return root
function search(root, val): if root is null: return false if val == root.val: return true if val < root.val: return search(root.left, val) else: return search(root.right, val)
