Data Structure

Heap / Priority Queue

Intermediate

A complete binary tree satisfying the heap property. In a max-heap every parent ≥ its children; in a min-heap every parent ≤ its children. Supports O(log n) insert / extract and O(n) build.

Insert: O(log n)
Extract: O(log n)
Peek: O(1)
Build: O(n)
Space: O(n)
🌳
Complete Binary Tree
All levels filled except possibly the last, which fills left-to-right. Stored efficiently in an array.
⬆️
Sift Up (Insert)
New element placed at end; bubble up by swapping with parent while heap property is violated.
⬇️
Sift Down (Heapify)
Root replaced with last element; push down by swapping with largest/smallest child until satisfied.
🏗️
Build Heap O(n)
Heapify all internal nodes bottom-up. More efficient than n individual inserts.

💻 Pseudocode
function buildHeap(arr): n = arr.length for i = floor(n/2)-1 downto 0: siftDown(arr, n, i) function siftDown(arr, n, i): target = i l=2i+1; r=2i+2 if l<n and arr[l] ≷ arr[target]: target=l if r<n and arr[r] ≷ arr[target]: target=r if target ≠ i: swap(arr[i], arr[target]) siftDown(arr, n, target)
function insert(heap, val): heap.push(val) // add to end i = heap.length - 1 while i > 0: parent = floor((i-1)/2) if heap[i] ≷ heap[parent]: swap(heap[i], heap[parent]) i = parent else: break
function extractRoot(heap): root = heap[0] heap[0] = heap.pop() // last → root siftDown(heap, heap.length, 0) return root
