Stack Operations

Beginner

A Stack is a Last-In, First-Out (LIFO) data structure. Elements are added and removed from the same end — the top. Visualize Push, Pop, Peek, and isEmpty operations step by step.

Push: O(1)
Pop: O(1)
Peek: O(1)
Search: O(n)
Space: O(n)
➕
Push
Add an element to the top of the stack. The new element becomes the new top.
➖
Pop
Remove and return the top element. The element below it becomes the new top.
👁️
Peek / Top
Look at the top element without removing it. Useful for checking what comes next.
📋
LIFO Principle
Last In, First Out — like a stack of plates. The last plate added is the first one removed.

💻 Pseudocode
class Stack: stack = [] def push(val): stack.append(val) def pop(): if isEmpty(): return null return stack.pop() def peek(): if isEmpty(): return null return stack[stack.length - 1] def isEmpty(): return stack.length == 0
